# Unity UI & Game Flow Prototype

Small Unity project focused on UI systems and game flow.

## Features
- Main Menu
- Options Menu (Audio & Fullscreen)
- Audio Manager (Music & SFX)
- Scene transitions with fade
- PlayerPrefs persistence

## Tech
- Unity 6.3
- C#
- TextMeshPro
- WebGL build

## Purpose
This project was created as a portfolio piece to demonstrate:
- UI implementation
- Scene management
- Audio handling
- Persistent settings
